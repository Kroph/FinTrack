# FinTrack

This project is a web application designed to help users track their income and expenses. It allows users to easily manage their financial flows, displaying information about amounts, dates, and categories of expenses, while helping them monitor their financial health.

## Features

- **Track Expenses and Income**: Add records of income and expenses with amounts, dates, and categories.
- **Reports**: The app automatically generates reports showing the dynamics of expenses and income, with the ability to filter by time.
- **Charts**: Visual representation of financial data using graphs and charts.
- **History**: View all previous records of expenses and income.
